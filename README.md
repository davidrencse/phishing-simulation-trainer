# Phishing Simulation Trainer

## Overview
A web application designed to simulate phishing attacks to educate users on recognizing and avoiding these threats. Users can experience realistic phishing scenarios and receive feedback on their responses.

## Features
- User registration and authentication
- Phishing simulation scenarios with different styles (emails, websites)
- Feedback and scoring system after each simulation
- Admin interface for creating and managing phishing scenarios
- Progress tracking and analytics for users

## Installation
1. Clone the repository.
2. Navigate to the project folder.
3. Run `npm install` to install dependencies.
4. Run `npm run dev` to start the development server.

## Deployment
To deploy this app to GitHub Pages, ensure the base URL in `vite.config.ts` is set correctly and follow the deployment instructions for GitHub Pages.
